 # Nota
 a binary clock build with C and raylib
 by 256k
 
 ```
o o o
o o o
o o o
o o o
o o o
o o o
```

*requires raylib*

to run:

`./build.sh && ./bin/nota`

NOTE: you can now optionally pass a circle radius value to nota to dynamically change circle width:

`./bin/nota 24`

(default circle width is 8)
